# AgentSafe: Problem, Solution, and How We Built It

AI agents are increasingly autonomous but lack standardized safety checks. Prompt injection allows malicious inputs that hijack agent behavior; harmful outputs include agents generating dangerous, illegal, or unethical content; unsafe tool use lets agents execute dangerous operations such as SQL injection or file deletion; and there is often no visibility when something goes wrong—no audit trail. Existing solutions like NeMo Guardrails and Guardrails AI are powerful but heavy, requiring YAML configs, Docker containers, or framework lock-in. AgentSafe solves this with a single decorator and zero config: you add `@safe` to your agent function and it wraps any LLM call with input and output safety checks. Safe prompts pass through; unsafe ones raise `SafetyViolation` before reaching the agent. Input scanning checks user prompts before they reach your agent; output scanning verifies agent responses before they reach users; tool call checking inspects tool calls before execution; and an intended-use scope guard blocks requests outside your app's purpose. Built-in observability includes metrics, traces, structured logs, and OpenTelemetry export, with violation callbacks for Slack, PagerDuty, or webhooks. We built AgentSafe as a lightweight, framework-agnostic Python library using httpx for HTTP, pydantic for validation, and the NVIDIA NIM API Catalog for safety inference. The core safety model is Llama-3.1-Nemotron-Safety-Guard-8B-v3, which classifies content across 23 safety categories in 9 languages via an OpenAI-compatible HTTP API—no NVIDIA SDKs or containers required. For intended-use scope guard we use Llama-3.1-Nemotron-Nano-VL-8B-v1, and for multimodal pipelines we integrate vision encoders and content-safety reasoning models. The library exposes a simple decorator API, a configurable `SafeAgent` wrapper, and streaming support via `safe_stream`. Observability is built in: every check produces a trace with timing, verdict, and categories, and we export to OpenTelemetry for Jaeger, Datadog, Grafana, and other backends. The site and docs are built with Next.js, React, TypeScript, Tailwind CSS, and React Flow for flow diagrams. AgentSafe is Apache 2.0 licensed and installable via `uv add agentsafe`.
